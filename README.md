# Metabolite-and-gut-microbial-composition-in-AML-Patients
Data associated with the manuscript - Host metabolism can produce many indoles and phenols independent of the microbiome

# Description
This dataset includes samples from three patients with acute myeloid leukemia who had fecal and serum samples taken before and after receiving broad-spectrum antibiotics. Serum metabolite measurements and fecal microbiome sequencing were completed. In each patient, gut microbial diversity and relative abundance of obligate anaerobic taxa were dramatically lower in stool samples during the broad-spectrum antibiotic treatment. One patient demonstrated loss of diversity, one recovery of diversity after stopping antibiotics, and one with recovery followed by loss upon re-initiation of antibiotics. Serum concentrations of the host-derived indole and phenol metabolites (indole-3-lactate, indole-3-acetate, indole-3-carboxylic acid, phenylpyruvate, phenyllactate, hydroxyphenyllactate) were not affected by microbiome compositional changes. In contrast, the microbiome-dependent metabolites indole-3-propionate and p-cresol sulfate were decreased in low-diversity, antibiotic-exposed samples.

# Data
<B>1. Metabolome:</B> This folder contains a csv file (AML_007-AML_013-AML_014-Stool-Blood_Metabolomics) with metabolite concentrations for all the stool and blood samples selected from the three patients (AML_007, AML_013, AML_014).

<B>2. Metagenome:</B> This folder contains 3 files with Metaphlan4 results for the samples for each patient in one file.

<B>3. Metadata file:</B> AML_007-AML_013-AML_014-Metadata.xlsx contains the metadata of the samples with sample_id, sample type, day of collection from the anchor event (start of chemotherapy), and alpha diversity for the stool samples.
